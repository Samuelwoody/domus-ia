## 📞 ¿NECESITAS AYUDA?

### **Preguntas Frecuentes:**

**Q: "¿Puedo usar esto en producción YA?"**  
A: Sí, pero solo con respuestas mock (demo). Para IA real necesitas backend.

**Q: "¿Cuánto cuesta desplegar?"**  
A: $5-20/mes (Railway) + $10-200/mes (OpenAI según uso)

**Q: "¿Cuánto tiempo toma activar backend?"**  
A: 1-2 horas siguiendo `ACTIVAR_BACKEND.md`

A: Para backend sí (básico). O puedes contratar a alguien.

**Q: "¿Los usuarios necesitan API key?"**  
A: NO (con backend). TÚ pagas OpenAI, ellos te pagan a TI.
